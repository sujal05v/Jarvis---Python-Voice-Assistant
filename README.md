# 🤖 Jarvis - Python Voice Assistant

A simple AI-powered voice assistant built using Python.
This project can listen to your voice commands, perform tasks like opening websites, playing music, and even respond using AI.

# 💡 Overview

This project was created while learning Python and exploring concepts like speech recognition, text-to-speech, and API integration.

It works similar to a basic version of assistants like Siri or Alexa.

# ⚙️ Features

* 🎤 Voice command recognition

* 🌐 Open websites like Google, YouTube, Facebook, LinkedIn

* 🎵 Play music from a custom library

* 📰 Fetch and read latest news headlines

* 🤖 AI-based responses using OpenAI API

* 🔊 Text-to-speech output

# 🛠️ Technologies Used

* Python

* SpeechRecognition

* pyttsx3 / gTTS

* pygame

* OpenAI API

* News API

# 📚 What I Learned

* Working with voice input in Python

* Converting text to speech

* Using APIs (OpenAI & News API)

* Handling real-time commands

* Building a basic AI assistant

# 🎯 Purpose

This is a beginner-to-intermediate level project built as part of my learning journey.
It helped me understand how real-world AI assistants work at a basic level.
